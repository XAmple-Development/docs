---
title: "Theme 404"
---

<Warning>
  This **MUST** be done before you can use the theme. If you do not do this, the
  theme will not work. You will have to put this in your DBD Config **NOT** your
  Theme Config!
</Warning>

```js
useTheme404: Boolean,
```

## Example Usage

```js
useTheme404: true;
```
