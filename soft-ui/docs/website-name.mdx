---
title: "Website Name"
description: "A [String](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) which is the name of your website and displayed in a lot of places."
---

```js
theme: SoftUI({
  websiteName: String,
});
```

## Example Usage

```js
theme: SoftUI({
  websiteName: "Assistants",
});
```

## Props

<ParamField path="websiteName" type="string" required>
  The name of the website
</ParamField>
